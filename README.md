# Pet4U - Animal Adoption Platform

Pet4U is an online platform for adopting animals, built using the MERN stack (MongoDB, Express.js, React.js, Node.js) and Redux for state management. The platform allows users to post animals for adoption, and it has a role-based access control system for users, sellers, and admins.

## Features

- **Role-Based System**: 
  - **User**: View available animals for adoption, submit applications for adoption.
  - **Seller**: Post animals available for adoption.
  - **Admin**: Manage users, seller posts, and ensure appropriate images and content.

- **User Authentication**: Secure authentication for different roles, ensuring proper access control.

- **Animal Listings**: Sellers can post animal listings, including images, descriptions, and other important details.

- **Admin Panel**: Admins can manage the platform by reviewing posts, managing user accounts, and ensuring content quality.

- **Redux State Management**: Centralized state management for seamless user experience across the platform.

## Tech Stack

- **Frontend**: React.js, Redux
- **Backend**: Node.js, Express.js, MongoDB
- **Authentication**: JWT (JSON Web Token) for secure login and registration
- **State Management**: Redux for handling application state
- **Database**: MongoDB (with Mongoose ORM)
